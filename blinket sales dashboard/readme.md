The Blinkit Sales Dashboard is an interactive data visualization project built to analyze and monitor sales performance for Blinkit (online grocery delivery platform). This dashboard helps in tracking key business metrics, identifying trends, and making data-driven decisions.

🚀 Features
📈 Sales performance analysis (daily, monthly, yearly)
🛍️ Category-wise and product-wise insights
🌍 Location-based sales breakdown
💰 Revenue and profit tracking
📦 Order volume and customer trends
🔍 Dynamic filtering for deeper analysis
🛠️ Tools & Technologies
Power BI – Data visualization and dashboard creation
Excel / CSV – Data source
DAX – Data modeling and calculations
📂 Dataset

The dataset includes:

Order details
Product categories
Sales amount
Customer locations
Dates and transaction data
